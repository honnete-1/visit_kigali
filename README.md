# Discover Kigali — Travel & City Guide Website

## Project Description
Discover Kigali is a multi-page website built using HTML and CSS that showcases some of the best places to visit and things to do in Kigali, Rwanda. The website provides visitors with useful information about popular destinations such as the Kigali Convention Centre, Nyamirambo, Rwandan cuisine, and the Kigali Genocide Memorial.

The project focuses on delivering a visually appealing, user-friendly, and informative experience for anyone planning to explore Kigali.

**Live Demo:** https://visit-kigali-omega.vercel.app/

---

## HTML Pages Included

This project consists of the following five pages:

- **index.html (Home Page)**  
  Provides an overview of Kigali and links to all destination pages.

- **kcc.html (Convention Centre Page)**  
  Highlights the Kigali Convention Centre, including events, features, and images.

- **nyamirambo.html (Nyamirambo Page)**  
  Showcases cultural experiences, local life, and activities in Nyamirambo.

- **cuisine.html (Rwandan Cuisine Page)**  
  Displays popular Rwandan dishes such as Isombe, Brochettes, and Rwandan Coffee.

- **memorial.html (Kigali Genocide Memorial Page)**  
  Provides educational and respectful information about the memorial site, including images and visitor guidelines.

## Design & UI Concept

The website uses a custom theme called:

**"Kigali Midnight & Gold"**

- Background: Clean light tones for readability
- Navigation: Dark midnight color for elegance
- Accent: Gold highlights representing opportunity and growth
- Hover effects: Subtle modern interactions

This design reflects the idea of **"Modern Kigali"** — a city that is clean, vibrant, and rapidly developing.
